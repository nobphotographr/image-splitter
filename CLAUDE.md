# Claude Code 指示書

## プロジェクト概要
画像を4分割するWebツール（X/Twitter縦長投稿用）

## 技術スタック
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4

## デザイン制約（必須）

**DESIGN.md を必ず参照すること。**

以下は絶対に守る：

1. **角丸**: `rounded-none` または `rounded`（4px）のみ。`rounded-lg` 以上は禁止
2. **影**: 一切禁止。`shadow-*` は使わない
3. **背景**: 白（`#FFFFFF`）または黒（`#0A0A0A`）。グレー背景＋白カード構図は禁止
4. **色**: テキスト黒/白、ボーダーグレー、アクセント1色のみ。青・緑などの多色禁止
5. **余白**: セクション間は `py-24`（96px）以上。詰め込み禁止
6. **カード**: 「箱に入れる」UIは避ける。セクションと余白で構成

## コード規約
- フォントは Geist Sans / Geist Mono を使用（layout.tsxで読み込み済み）
- CSS変数でカラートークンを管理
- コンポーネントはシンプルに、過度な抽象化は避ける
