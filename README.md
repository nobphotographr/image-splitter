# 画像4分割ツール

X（Twitter）の縦長投稿用に画像を4分割するWebアプリです。

## 特徴

- 画像を縦に4等分
- ドラッグ&ドロップ対応
- ブラウザ上で完結（データ保存なし）
- レスポンシブデザイン
- ダークモード対応

## 使い方

1. 画像をドラッグ&ドロップするか、ファイルを選択
2. 自動的に4分割されます
3. 個別にダウンロードするか、全てダウンロード
4. Xの投稿画面で上から順番に4枚を選択

## 開発

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いて確認できます。
